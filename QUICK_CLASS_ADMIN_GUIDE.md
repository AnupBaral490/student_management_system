# Quick Guide: View Students & Teachers in Class Admin

## How to View Students and Teachers for a Class

1. **Access Admin Panel**
   - Go to: `http://your-domain/admin/`
   - Login with admin credentials

2. **Navigate to Classes**
   - Click on **Academic** in the left sidebar
   - Click on **Classes**

3. **Select a Class**
   - Find "BIM 7th Semester - Year 4, Sem 7 - A" (or any class)
   - Click on it

4. **View Information**
   - You'll see all enrolled students in the "Enrolled Students" section
   - You'll see all assigned teachers in the "Teacher Assignments" section

## Quick Actions

### Add a Student to the Class
1. Scroll to "Enrolled Students" section
2. Click "Add another Enrolled Student"
3. Select student from dropdown
4. Check "Is active"
5. Click "Save"

### Add a Teacher to the Class
1. Scroll to "Teacher Assignments" section
2. Click "Add another Teacher Assignment"
3. Select teacher and subject
4. Click "Save"

### Remove a Student or Teacher
1. Find the student/teacher in the list
2. Check the "Delete" checkbox
3. Click "Save"

## What You'll See

- **Student Count**: Total number of active students in the class
- **Teacher Count**: Total number of teachers assigned to the class
- **Student Details**: Name, ID, enrollment date, active status
- **Teacher Details**: Name, subject they teach, academic year

That's it! Now you can easily manage students and teachers for each class directly from the class admin page.
