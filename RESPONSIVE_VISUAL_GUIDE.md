```html
<div class="btn-group">
  <button class="btn btn-primary">Action</button>
  <button class="btn btn-success">Action</button>
</div>
```
- Desktop: Horizontal
- Mobile: Vertical

### Grid Pattern
```html
<div class="row">
  <div class="col-lg-4 col-md-6 col-12">Content</div>
</div>
```
- Large: 3 columns
- Medium: 2 columns
- Small: 1 column

This visual guide helps understand how the system adapts to different screen sizes!
